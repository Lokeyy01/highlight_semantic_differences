ImageMaster 9 - Compatibility Guide 1 Introduction and overview

1 Introduction and overview

1.1 Conventions in this guide

The supported platforms and third-party software modules given within this guide are the ones whichImageMaster has been approved for. This does not mean that ImageMaster will not work with otherproducts or versions. If in doubt, contact T-Systems to get more information. The following con-ventions apply:

ImageMaster supports the specified third-party software version. If there are minor versions orpatches available for this version, which are recommended by the manufacturer for important errorcorrections and component upgrades, you should use the latest stable minor / patch version withImageMaster.

For a general product overview, which lists further documentation of ImageMaster 9, see the productdescription [T Product Description]. For the latest information on the current release also see therelease notes [T Release Notes].

1.2 Changes in supported environments

Only major changes are summarized here. A beginning or end of support is not listed for each minorversion of any 3rd-party component. For such details check the respective main sections in this doc-ument.

ImageMaster 9 - Compatibility Guide 3 Mobile device support

3 Mobile device supportThe ImageMaster WorkplaceClient supports mobile devices in various constellations. Some have expli-citly been tested as stated below.

Mobile device support for other ImageMaster components is available on demand. This refers to theContent Services for E-Mail and SharePoint and includes the statements on iOS / Android devices withbrowsers (Chrome / Safari) or native apps such as Outlook, but also Windows 10 “touchable” PCs.

The following mobile devices have explicitly been tested with the WorkplaceClient:

■ Android tablet: Samsung Galaxy Tab A 10.1 (2019) / Android 11 / Chrome

■ Android phone: Samsung Galaxy S20 / Android 12 / Chrome

■ iOS phone: iPhone XR / iOS 15.7.1 / Safari

■ iOS pad: iPad Air2 / iOS 15.7.1 / Safari

On mobile devices the quality and the layout of displayed content depends on a complex con-stellation of factors involving the viewport size. To achieve a mobile-friendly representation it isstrongly recommended using devices which are not older than 4 years.

In the beginning of 2020 Microsoft changed the strategy and does not provide access to mail add-insof Outlook in mobile browsers. Therefore, also the ImageMaster MailApp is no longer displayed onmobile browsers.

ImageMaster 9 - Compatibility Guide 4 ImageMaster Content Services for E-Mail

4 ImageMaster Content Services for E-MailThe ImageMaster Content Services for E-Mail are tailored to Microsoft Exchange Server envir-onments. In such environments there is a close interconnection between Microsoft products such asMS Office/Outlook, MS Exchange Server, and MS Active Directory Services. It has been observed inpractice that unexpected behavior can sometimes result from an untested combination of theseproducts, and it can also be caused by Microsoft updates.

In e-mail environments with Microsoft Exchange and Outlook, items such as e-mails and calendarentries are categorized into message classes like “IPM.Note”, which represents a standard class of ane-mail. For a list of approved message classes see chapter “Approved message classes” in [SM Mail].

Windows 10 S with its Outlook version from the Windows Store is not supported by ImageMaster.

■ Exchange Online

ImageMaster 9 - Compatibility Guide 6 ImageMaster product lifecycle and support models

6 ImageMaster product lifecycle and support modelsSupport duration

ImageMaster is released with two different models:

■ The “innovation release” is published in quarter 2.

■ The “long-term support release” is published in quarter 4.

Both releases are full-fledged ImageMaster releases, which provide enhancements and new func-tionality. These are fully tested and available for productive use. The only difference between both isthe support duration:

■ The innovation release comes with ≥12 months of standard software support lasting always until30.6. of the following year.

■ The long-term support release comes with ≥36 months of standard software support lastingalways until 31.12. three years later. The standard support is followed by the extended support of24 months lasting until 31.12. two years later.

Third-party software support

Support for new major versions of third-party software is introduced with new ImageMaster releases.T-Systems expects that customers will remain on a supported environment. In the event that loosely-coupled depending third-party software(a) is no longer supported(b) by its vendor, T-Systems may belimited in its ability to provide support and may require you to upgrade to a newer ImageMasterrelease to reproduce the issue in a supported environment and to benefit from continued assistance.(a) Examples of such third-party software in an ImageMaster environment include the operating sys-tem, the application server, storage types, and the full-text engine.(b) The lack of support can also refer to a specific version of this software.
